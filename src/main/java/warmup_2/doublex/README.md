<h3>PROBLEM:</h3>

**Given a string, return true if the first instance of "x" in the string is immediately followed by another "x".**
#

<h3>Exemples:</h3>

- _doubleX("axxbb")_ → **true**
- _doubleX("axaxax")_ → **false**
- _doubleX("xxxxx")_ → **true**
